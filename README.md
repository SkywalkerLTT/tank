# 🎮 坦克大战 (Tank Battle)

经典 **Battle City** 风格坦克大战 HTML5 游戏！

## 玩法

| 操作 | 按键 |
|------|------|
| 移动 | `↑` `↓` `←` `→` |
| 射击 | `空格键` |
| 暂停 | `P` |

## 特性

- 🎯 3 个关卡，难度递增
- 🤖 智能敌人 AI（会追踪玩家！）
- 🧱 可破坏的砖墙 + 坚不可摧的钢墙
- 🦅 需要保护的基地（鹰）
- 💥 粒子爆炸特效
- ❤️ 生命系统 + 计分
- 📱 触摸屏支持
- 🏆 自动进入下一关

## 在线试玩

直接打开 `index.html` 即可畅玩！

---

### 在本地运行

```bash
# 克隆仓库
git clone https://github.com/SkywalkerLTT/tank.git

# 直接打开
cd tank
start index.html
```
